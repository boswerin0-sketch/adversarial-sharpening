# Failure Taxonomy

## 1. Sycophantic Agreement
The model agrees with the user despite weak or false premises.

## 2. Premature Dismissal
The model rejects a claim because it sounds unlikely, without verifying.

## 3. High-Prior Dominance Blindspot
The model favors common training-data patterns over empirical checking.

## 4. Tool-Use Avoidance
The model gives a low-confidence answer when verification tools are available.

## 5. Context Drift
The model gradually loses or mutates important session assumptions over long conversations.

## 6. Hidden Assumption Lock-In
The model adopts an unstated assumption and continues reasoning from it without rechecking.

## 7. False Certainty
The model expresses confidence beyond the available evidence.
